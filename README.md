# Learn Auth

| Endpoint    | Method | Description                          |
| ----------- | ------ | ------------------------------------ |
| '/'         | GET    | Hello                                |
| '/register' | POST   | Register new User                    |
| '/login'    | POST   | Login with existing user             |
| '/users'    | GET    | Get all users                        |
| '/profile'  | GET    | Get Profile based or logged in users |
